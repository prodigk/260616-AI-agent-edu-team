# Role Agent Instructions

이 폴더는 우측 채팅 답변 담당 에이전트의 답변 지침을 관리합니다.

`src/role_agents.py`는 라벨, 키워드, 담당 도메인, 라우팅 메타데이터를 정의하고,
각 에이전트의 실제 답변 지침은 이 폴더의 Markdown 파일에서 읽습니다.

- `marketing.md`: 마케팅 담당
- `customer_cs.md`: 고객 CS 담당
- `inventory.md`: 재고관리 담당
- `sales_product.md`: 매출 상품 담당
- `product_planning.md`: 제품 기획 담당
- `product_lead.md`: 제품총괄
